# About
This is a fork of Luke Smith's build of DWM, which has been slightly modified. 
It just has different keybindings and colors. 

## Installation

```bash
git clone https://github.com/LukeSmithxyz/dwm.git
cd dwm
sudo make install
```

There is also a `PKGBUILD` usable on distributions with pacman. Run `makepkg -si` instead of `sudo make install`.
